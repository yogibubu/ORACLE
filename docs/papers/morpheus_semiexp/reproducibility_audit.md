# MORPHEUS Paper Reproducibility Audit

This audit tracks which numerical results in the MORPHEUS manuscript can be
regenerated from the current MATRIX repository.

## Reproducible From Committed MATRIX Commands

- Main benchmark summary and planar-pair diagnostics:

  ```bash
  python -m matrix semiexp-benchmark \
    --snapshot benchmarks/semiexp_msr/golden/semiexp_paper_regression.json \
    --outdir docs/papers/morpheus_semiexp/generated
  ```

  This regenerates `paper_benchmark_summary.csv`,
  `paper_planar_pair_diagnostics.csv`, `benchmark_summary.tex`, and
  `planar_pair_diagnostics.tex` byte-for-byte.

- Single-run examples with committed primary inputs:

  ```bash
  python -m matrix semiexp \
    --job benchmarks/semiexp_msr/inputs/nitrobenzene/nitrobenzene_all_msr.inp \
    --outdir working/semiexp/nitrobenzene/run_pair_AB \
    --rotational-components AB

  python -m matrix semiexp \
    --job packages/matrix-morpheus/examples/semiexp/maleic_anhydride/maleic_anhydride_ensemble.mse.toml \
    --outdir working/semiexp/maleic_anhydride

  python -m matrix semiexp \
    --job packages/matrix-morpheus/examples/semiexp/phthalic_anhydride/phthalic_anhydride_predicates.mse.toml \
    --outdir working/semiexp/phthalic_anhydride

  python -m matrix semiexp \
    --job packages/matrix-morpheus/examples/semiexp/succinic_anhydride/succinic_anhydride_fixed_h.mse.toml \
    --outdir working/semiexp/succinic_anhydride_fixed_h

  python -m matrix semiexp-ensemble \
    --job packages/matrix-morpheus/examples/semiexp/anhydrides_ensemble/anhydrides_ensemble.mse-ensemble.toml \
    --outdir working/semiexp/anhydrides_ensemble

  python -m matrix semiexp-ensemble \
    --job packages/matrix-morpheus/examples/semiexp/glycine_ensemble/glycine_conformers_synthon.mse-ensemble.toml \
    --outdir working/semiexp/glycine_ensemble_synthon

  sh packages/matrix-morpheus/examples/semiexp/testosterone/run_testosterone_cc_class.sh

  sh packages/matrix-morpheus/examples/semiexp/testosterone/run_testosterone_cc_co_classes.sh

  sh packages/matrix-morpheus/examples/semiexp/testosterone/run_testosterone_predicates.sh
  ```

  The testosterone scripts produce the one-class C--C baseline, the disjoint
  C--C/C--O class model, and the weighted QC-predicate model reported in the
  manuscript and supporting information.

## Reproducible From Committed Snapshot

The paper benchmark includes production rows whose final historical run
directories are not committed. Their numerical values are nevertheless
versioned in `benchmarks/semiexp_msr/golden/semiexp_paper_regression.json` and
are regenerated into the manuscript tables by `semiexp-benchmark`.

This category currently includes the p-EBN production row and the exact
Glycine I/Glycine II `pcs4_auto`/`dpcs3_auto` rows used by the manuscript
tables.

## Remaining End-to-End Gaps

- p-EBN legacy Z-matrix input is now parsed, including named Z-matrix GIC
  definitions and legacy relation constraints, but the direct default run from
  `benchmarks/semiexp_msr/inputs/legacy_msr_import/p-EBN.msr.inp` does not yet
  reproduce the production row. It currently terminates with no active totally
  symmetric parameters. The production workflow needs an explicit MATRIX job
  wrapper or restored run manifest describing the same symmetry/model options
  used for the paper row.
- Several paper-specific generated tables and figures, for example
  `glycine_I_qc_rotational_comparison.tex`,
  `glycine_I_pcs4_geometry_comparison.tex`,
  `p_ebn_geometry_parameters.tex`, and the figure PDFs, are committed as paper
  assets but do not yet have one command per artifact in MATRIX.

Before submission, every non-static paper table should either be regenerated by
a committed command or explicitly marked as a frozen historical regression
asset with its input/run manifest.
