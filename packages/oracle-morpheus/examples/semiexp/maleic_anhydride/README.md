# Maleic anhydride semiexperimental refinement

Data are transcribed from Vogt, Demaison and Rudolph, Struct. Chem. 2011, 22, 337-343.

- `parent.xyz` is a planar C2v starting geometry reconstructed from the reported rse parameters in Table 1.
- `isotopologues.toml` contains Table 5 B0 constants, the tabulated MP2/cc-pVTZ rovibrational corrections, and electronic corrections inferred from the reported Be values.
